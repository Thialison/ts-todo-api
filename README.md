### Todo API (Typescript)

RESTFul API para você criar os seus TODOS. Usando os principios do SOLID

#### Tecnologias
* Node
* Typescript
* Express
* MongoDB

#### Dependências
* Nodejs
* Docker
* Docker Compose


#### Executando a API

Suba o container do mongoDB
```
docker-compose up -d
```

Execute os testes 
```
yarn test
```

Inicie a aplicação
```
yarn dev
```
